# Getting Started with LenovoQiraDRJ

## Installation
1. Download the VSIX extension file
2. Double-click the file to install it in Visual Studio
3. Restart Visual Studio to complete the installation

## Usage
1. Open a Git repository in Visual Studio
2. Use the Git helper features from the menu
3. View code analysis results in the Error List
4. Open the Summary Tool Window from the View menu

## Features
- Git repository analysis
- Code analysis with custom rules
- Summary tool window for repository information
